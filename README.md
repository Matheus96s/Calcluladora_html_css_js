# Calcluladora_html_css_js
Calculadora Simples do Matheus para Principiantes
O Repositorio tem como objetivo simular um projeto de um calculadora que está sendo desenvolvido em equipe, contando com diferentes diversos colaboradores. 
O proposito é demostar como seria desenvolver um projeto em equipe, demostrando o trabalho em equipe em um mesmo repositorio, como uma empresa, por exemplo, o mesmo não se atenta tanto a questões técnicas como os codigos presentes na programação do script e até mesmo à base de dados, servindo apenas como um auxilio ao proposito principal.
